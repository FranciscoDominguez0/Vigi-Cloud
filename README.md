# VigiCloud

This repository contains the VigiCloud Nextcloud distribution with custom apps, branding, themes, and configuration.

## Structure

- `docker-compose.yml` - Docker Compose definition
- `Dockerfile` - Build instructions for the Nextcloud environment
- `.env` - Environment variables
- `nextcloud/` - Nextcloud source tree
- `custom_apps/` - Custom applications
- `themes/` - Custom theme assets
- `branding/` - Logos, login and email templates
- `config/` - Configuration files
- `skeleton/` - Default files for new users
- `scripts/` - Backup, restore and install scripts
- `nas/` - NAS mount or storage data
- `backups/` - Database and file backups
- `mariadb/` - MariaDB data
- `redis/` - Redis data
